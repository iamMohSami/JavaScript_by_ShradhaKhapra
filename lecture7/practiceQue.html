Homework : Study about appendChild() and removeChild()

appendChild() and removeChild() are two DOM (Document Object Model) methods in JavaScript used to manipulate the structure of HTML documents by adding or removing elements. The appendChild() method is used to append a node as the last child of a specified parent node. It adds a node to the end of the list of children of a specified parent node. The removeChild() method is used to remove a specified child node from the DOM. It removes a child node from the parent node.

Differences bw .append() and .appendChild() || .remove() and .removeChild()

appendChild() vs. .append()
Return Value: appendChild() returns the appended child node. This can be useful if you want to keep a reference to the appended node. .append() does not return a value.
Multiple Arguments: appendChild() accepts a single node as an argument. If you want to append multiple nodes, you need to call it multiple times. .append() accepts multiple arguments and appends each of them. This can make it more concise when dealing with multiple elements.


removeChild() vs. .remove()
Return Value: removeChild() returns the removed child node. Similar to appendChild(), this can be useful if you want to keep a reference to the removed node. .remove() does not return a value.
Syntax: removeChild() requires a reference to the parent node and the child node to be removed. .remove() can be called directly on the child node without requiring a reference to the parent.
